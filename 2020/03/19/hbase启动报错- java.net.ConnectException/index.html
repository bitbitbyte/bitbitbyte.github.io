问题初步排查 : 端口是否可以正常连接

问题排查

node101:9000端口无法正常连接, 有如下几种可能:

  1. node101设置了防火墙
  2. hadoop未正常启动
  3. node101不为namenode所在主机
  4. namenode的端口号不为9000

问题解决

查看hadoop的core-sit.xml的文件系统配置信息如下

fs.defaultFS: 接收Client连接的RPC端口，用于获取文件系统metadata信息。
